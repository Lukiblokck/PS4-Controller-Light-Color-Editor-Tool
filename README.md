# PS4 Controller Light Color Editor Tool

A simple Windows Forms application to customize the light bar color of your PS4 DualShock controller. 

## Features

- **Choose Any Color**: Open a color picker and set your favorite color to the PS4 controller light bar.
- **Rainbow Mode**: Activate a continuous rainbow effect on your controller light.
- **Live Preview**: See the selected color on the panel in the application before sending it to the controller.
- **Quick Access Links**:
  - [Developer's Website](https://lukiblokck.github.io)
  - [Creative Commons License](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## How It Works

1. Connect your PS4 controller via USB.
2. Open the application.
3. Choose a color using the **"Choose the color"** button or enable rainbow mode.
4. The color will immediately update on your controller's light bar.

## Requirements

- Windows OS
- .NET 6.0 or later
- USB connection for DualShock 4

## Installation

1. Download the `release`.
2. Unzip the `.zip`.
3. Run the `.exe` from the `PS4 Controller Light Color Editor Tool` folder.
4. Now enjoy!!!
